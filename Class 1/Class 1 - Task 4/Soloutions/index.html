Hello World

Most People's First Intro to Programming

"Hello World" is usually the first thing someone learns when practicing coding, but where does it originate?

The origin of using "Hello World" as a test phrase when writing code dates back to 1978 in the book "The C Programming Language" by Brian Kernighan and Dennis Ritchie.

The phrase is still used today in an homage to the early days of digital computer programming and if most often used to illustrate a languages basic syntax (the way the language is written for the computer to understand)
